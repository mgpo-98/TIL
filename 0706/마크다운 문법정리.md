- 문법

  - 제목

  #(띄어쓰기) (제목) 을하면 (예: # 예시문) 큰 제목을 만들수 있다.

  '#'의 갯수에 따라 제목의 크기가 점점 줄어들어 소제목도 만들수 있다.

  - 리스트

    -(띄어쓰기) (엔터) 를 하면 위와 같이 리스트별로 나눌수 있다.

    리스트안에 리스트도 만들어 더 세분화 하여 나눌수있다

  - 코드 블록 

    코드를 작성할때 (```)backtick,(pyton, html 등...)를 사용하여 저 안에 코드를 작성하면 블록이  생성되어

    코드를 보기 편하다

    ```python
    print('hello')
    # 주석
    ```

    ```html
    print('dd')
    <h1>
        <!--주석 -->
    ```
